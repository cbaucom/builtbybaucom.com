## ARIA Labels

ARIA labels help screen readers understand your content:

```html
<button aria-label="Close menu">
  <svg>...</svg>
</button>
```

## Keyboard Navigation

Ensure your application is fully navigable via keyboard:

```typescript
function handleKeyPress(event: KeyboardEvent) {
  if (event.key === 'Enter' || event.key === ' ') {
    // Handle activation
  }
}
```

## Color Contrast

Maintain sufficient color contrast for text:

```css
/* Good contrast */
.text-primary {
  color: #2563eb;
  background: #ffffff;
}
```
