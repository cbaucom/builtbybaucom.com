# Portfolio Website

A modern, responsive portfolio website built with Next.js and TypeScript. Features a clean design, blog functionality, and project showcase.

## Features

- Responsive design
- Dark mode support
- Blog with MDX
- Project showcase
- Contact form
- SEO optimization

## Technology Stack

- Next.js
- TypeScript
- Styled Components
- MDX
- React Hook Form

## Implementation Details

The website uses Next.js for server-side rendering and static site generation, providing excellent performance and SEO benefits.
